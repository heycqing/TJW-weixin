## nodeJs开发微信公众号

> 使用方法

### 运行环境（线上和测试）

1. 在**测试环境**运行，可以通过**localtunnel**来进行外网映射
 
    + 1.1 安装**localtunnel**

    ```npm install -g localtunnel ```
    
    + 1.2 启动服务

        - 1.2.1 在根目录启动 `app.js `

            ``` node app.js ```

        + 1.2.2 然后，启动 映射服务 

            ```lt --port 5544(端口号) ```

        + 1.2.3 在微信测试号配置好 **链接**

    + 1.3 进入测试公众号进行测试；
